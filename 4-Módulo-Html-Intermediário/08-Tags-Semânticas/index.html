Deck Controle

Um deck controle é um deck que visa ganhar a partida com o esgotamento das condições de vitória do oponente

Uma das classes que mais se destaca nesse quesito é o Guerreiro, com mecanismos de suportar o dano causado pelo oponente até que não sobrem mais recursos ao oponente

Para aprender na prática esse conceito, confira esta gameplay do Fibonacci

Deck Aggro

Um deck aggro é um deck que visa matar o adversário antes que ele consiga tirar valor de suas cartas.

O caçador é a classe que mais se destaca nesse quesito

Para aprofundar os conhecimentos desta classe na prática, escute Esta música explicativa.
